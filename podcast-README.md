```
  💡 Ideia & Conceito
  (ChatGPT — Nicho, Persona, Tom)
         │
         ▼
  🏷️ Nome & Tagline
  (ChatGPT — 10 opções → "CodeSpark")
         │
         ▼
  📝 Roteiro Completo
  (ChatGPT — Estrutura + Conteúdo + Marcadores de gravação)
         │
         ▼
  🖼️ Capa Visual
  (Midjourney v6 — Prompt detalhado → Arte final)
         │
         ▼
  🔊 Narração com IA
  (ElevenLabs — Voz humanizada em português brasileiro)
         │
         ▼
  ✂️ Edição
  (Descript / Audacity — Trilha, efeitos, masterização)
         │
         ▼
  📢 Podcast Finalizado
```

## 💡 Destaques da Engenharia de Prompt

Este projeto aplicou **7 categorias de prompts** ao longo do processo criativo:

**1. Prompt de Persona**
> Definir o papel da IA antes da tarefa — "Você é um roteirista experiente de podcasts de tecnologia brasileiros."

**2. Prompt de Contexto Rico**
> Fornecer informações completas sobre público, tom, formato e objetivo antes de qualquer instrução.

**3. Prompt de Formato Especificado**
> "Crie uma tabela com 5 colunas: bloco, conteúdo, duração, tom e CTA."

**4. Prompt de Criatividade Guiada**
> Para o Midjourney: descrição visual detalhada com estilo, cores, composição e parâmetros técnicos.

**5. Prompt de Iteração**
> Refinamento em múltiplas rodadas: "Bom começo, agora adicione mais dados e torne o tom 20% mais próximo."

**6. Prompt para Voz (ElevenLabs)**
> Instruções de entonação inseridas no texto como marcadores: [PAUSA], [ÊNFASE], [RESPIRAÇÃO].

**7. Prompt de Divulgação**
> Roteiro específico para posts em LinkedIn, descrição no Spotify e artes de redes sociais.
